**What is the general structure of the project?**

The repository has the following top-level directories:
1. `c/`: The C-based engine implementation.
2. `web/`: A pure OpenAI-API client, community-maintained (not a runtime dependency).
3. `tools/`: Offline conversion, fixtures, and benchmarks.
4. `scripts/`: Long-running conversion helpers.

The project uses a flat, readable structure with minimal dependencies. Existing commands run from the `c/` directory continue to work unchanged.

**Why is it called "colibrì"?**

The name refers to the hummingbird (Colibri), which weighs only a few grams, hovers in place, and visits a thousand flowers a day. This engine keeps a 744-billion-parameter giant alive on hummingbird rations: 25 GB of RAM, twelve CPU cores, and a lot of disk patience.
